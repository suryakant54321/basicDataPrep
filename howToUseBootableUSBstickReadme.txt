Title: How to use bootable USB stick on any working computer 
==============================================================================

Suggested but not necessairy hardware requirments
	1. RAM > 2 Gb (higher is better)
	2. Processor > Intel Core 2 Duo (higher is better)
	3. No hard drive limiatation (actually you don't need it unless you have 
		large data storage needs)
------------------------------------------------------------------------------

------------------------------------------------------------------------------

USB booot Instructions 		
	1. Access BIOS of your computer (may come by pressing F2 key while booting)
	2. In case of windows 10 / 8 user may face problems in rebooting the system
		if you face such problems try pressing power button for long time to 
		switch off the system and start again
	3. If you are in BIOS menu
		a. enable USB boot / boot from usb
		b. provide first priority for USB boot
	4. save and exit the BIOS settings
	5. Now computer attached with bootable USB must boot from USB i.e. UbuntuMATE OS
	6. If Windows is still booting. Repeat steps from 1 to 5
		or check the BIOS settings manual of your computer
